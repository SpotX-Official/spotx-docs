SpotX - Experimental Features

List of active experimental features provided in SpotX

For Windows​

Ban/unban artists​

Feature from: Spotify

Description: Enable context menu action to ban/unban artists

Top search bar​

note

This feature is disabled by default

Feature from: Spotify

Description: Show global nav bar with home button, search input and user avatar

To activate add input parameter -topsearchbar

Developer tools​

note

This feature is disabled by default

Feature from: Spotify

Description: Activates devtools which have been limited to use by Spotify employees only, to enable them, simply go to the bottom of the main settings and toggle the switch. After an automatic reload, you will have access to full development tools, including the debug window, cosmos debugger, and developer utils.

To activate add input parameter: -devtools

warning

Please use this only if you understand clearly what you are doing. SpotX provides access to these tools for familiarization purposes and is not responsible for your actions.

note

Works from 1.1.98 and higher

Plus button​

note

This feature is disabled by default

Feature from: Spotify

Description: The plus button replaces the heart button, you can save and select destinations for songs, playlists, or podcasts

To activate add input parameter -plus

Smart shuffle​

Feature from: Spotify

Description: Smart Shuffle adds recommended tracks to the play queue only in personal playlists

Funny progress bar​

note

This feature is disabled by default

Feature from: Spotify

Description: Funny progress bar.

To activate add input parameter -funnyprogressBar

Dynamic audio normalization​

Feature from: Spotify

Description: Actively adjust the volume by leveling out the difference between quiet and loud parts of the audio.

Audio cache clear limit changes​

note

This feature is disabled by default

Feature from: SpotХ

Description: Change limit for clearing audio cache.

-cache_limit [number] - from 500 to 20000

Example: set the cache limit to 700 MB -cache_limit 700 or -cl 700

note
  • By default for all users the limit is set to 10 GB
  • Cleaning affects only the audio cache, downloaded tracks, podcasts will not be affected

The history of listening to songs​

note

This feature is disabled by default

Feature from: SpotХ

Description: The history of listening to the songs of your songs for use in the Goofy playlist builder.

Two input parameters are used for activation:

  • -urlform_goofy
  • -idbox_goofy

Detailed instructions on how to set up are here, instructions in Russian, use a translator.

Hiding Ad-like sections from the homepage​

note

This feature is disabled by default

Feature from: SpotX

Description: an attempt to keep only those sections on the main page that recommend music based on your listening.

To activate add input parameter -adsections_off

If you have your own thoughts about which sections to hide and which not, then write here,
if you still have a hidden section, then send its id here using this instruction.


full list of hidden sections:

  • Popular Albums / Artists
  • Popular new releases
  • Best of Artists / Tracks
  • Best of songwriters
  • Fresh New Music
  • The state of music today
  • Mood
  • Today's biggest hits
  • Throwback
  • Trending now
  • Popular radio
  • Try something else
  • Featured Charts
  • The best of 2022
  • The Best Christmas Music
  • Happy Holidays

Static themes for lyrics​

note

This feature is disabled by default

Feature from: SpotX

Description: Spotify in the original client uses automatic colors for lyrics, the color is selected based on the album art, sometimes this leads to colors that are not very pleasant to look at, this feature adds static themes.

list of available themes:

to activate a static theme use the input parameter lyrics_stat
so for example we activate theme blueberry: -lyrics_stat blueberry

You can also add your own theme, just send your colors as specified in this template.

Almost all topics were made by @IshanJ25 many thanks to him
also thanks for the idea @CanYouJustWorkPlease

Unlock in device picker​

Feature from: SpotX (Spotify Premium)

Description: Unlock in device picker.

note

Not all devices can be unlocked, some devices check the premium subscription through the server.

BeforeAfter

Full screen mode​

Feature from: SpotX (Spotify Premium)

Description: Turns on full screen mode when listening to music

New home structure and navigation​

Feature from: Spotify

note

New theme force activated for everyone since 1.2.14

Description: New Spotify cover

To enable use the input parameter -new_theme
